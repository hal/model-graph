WildFly Model Graph

Analyse the WildFly management model using a graph database.

This site provides links to Neo4J databases containing the management model of WildFly 9, 10 and 11. The links below open the Neo4J browser for the given WildFly version. For more information about WildFly Model Graph, please see the project's README.

Please note:
Per default the Neo4J browser suggests to connect to the same host it was loaded from. The databases below use a different host (see below). Pay extra attention to specify port 80 as host (otherwise Neo4J falls back to the default port 7687). If the connection form doesn't show up automatically, use the :server connect command to trigger the connection form. Use neo4j as username and password.
